Politique de confidentialité

Dernière mise à jour : [Date]

1. Introduction

Votre vie privée est importante pour nous. Cette politique de confidentialité décrit comment [Nom de l’application] (“nous”, “notre”, “nos”) collecte, utilise et partage les données des utilisateurs lorsque vous utilisez notre application.

2. Données collectées et finalités

Nous pouvons collecter les éléments suivants : données techniques (type d’appareil, système d’exploitation), identifiants publicitaires, données de localisation approximative, etc., afin de :

  • Afficher des annonces personnalisées ou non personnalisées via nos partenaires (ex : annonces via AdMob) ;
  • Améliorer l’expérience utilisateur, analyser les performances de l’application ;
  • Respecter nos obligations légales.

3. Consentement & annonces personnalisées

Pour les utilisateurs situés dans l’Espace Économique Européen (EEE), nous utilisons le SDK User Messaging Platform (UMP) pour recueillir votre consentement avant de charger les annonces personnalisées. Si vous refusez, seules des annonces non personnalisées seront affichées.

4. Partage des données

Nous ne vendons pas vos données. Nous pouvons partager certaines données avec nos partenaires publicitaires, mais uniquement dans le respect de vos choix (personnalisées vs non personnalisées).

5. Vos droits

Vous pouvez à tout moment retirer ou modifier votre consentement via les paramètres de l’application ou en nous contactant à contact@tonapp.com. Vous avez également le droit d’accéder à vos données, de les rectifier ou de demander leur effacement.

6. Sécurité

Nous prenons des mesures raisonnables pour protéger vos données, mais aucun système n’est totalement infaillible.

7. Modifications

Nous pouvons mettre à jour cette politique périodiquement. Nous vous informerons des modifications via l’application ou par une mise à jour visible de la date.
